📝 All Config Options

VariableRequiredDefaultDescription
PORT Optional 3005 Port the auth server listens on
BETTER_AUTH_SECRET Required — 32+ char random string for encryption. Generate: openssl rand -base64 32
BETTER_AUTH_URL Required — Public URL of this auth service (e.g. https://your-app.railway.app)
DATABASE_URL Required — PostgreSQL connection string
FRONTEND_URL Required http://localhost:5173 Your frontend origin (for CORS)

⏱️ Session Timeline Visual

How Better Auth manages sessions:

User Logs In
Fresh Zone (30 min)
No DB writes during this window
Active User
Session extends +24h on activity
Refresh triggered every 1 hour
User Goes Idle
Session expires after 24h idle
Must sign in again
💡 This follows the video's best practices: Short session windows + automatic rotation. If a token is stolen, it's only valid for max 24 hours (or less if the user logs out).

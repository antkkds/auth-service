📖 How This Works

1
User signs up → POST /api/auth/sign-up/email with name, email, password
2
Better Auth creates session → Returns session cookie (short-lived, auto-rotating)
3
Your frontend uses session → Call GET /api/auth/get-session to verify user
4
Session auto-refreshes → If user is active, session extends. Idle = expires.
